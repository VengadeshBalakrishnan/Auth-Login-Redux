### Environment Variables
```
NODE_ENV = development
PORT = 5000
MONGO_URI = 'insert_your_mongodb_uri'
JWT_SECRET = "login-auth-secre"
```

### Install Dependencies
Backend & Frontend
```
  npm install
  cd frontend
  npm install
```

### Available Scripts
```bash
  # Both frontend (:3000) & backend (:5000)
  npm run dev

  # Backend
  npm run server

  # Frontend
  npm run client
```
